# MijnBureau OpenProject
OpenProject image with plug-in added to enable OIDC for MijnBureau.

Similar as for openDesk, the team of OpenProject created a plugin for MijnBureau which allows to use of OIDC. With this plugin we will be able to create an OIDC provider with env variables.

→ https://github.com/opf/openproject-mijn_bureau

## Related information
* Docs of how to use the plugins for OpenProject → https://www.openproject.org/docs/installation-and-operations/installation/docker/#openproject-plugins
* Some examples for configuration via environment variables are here → https://www.openproject.org/docs/system-admin-guide/authentication/openid-providers/#configuration-using-environment-variables
